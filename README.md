# The Living Olive

A mobile app for daily spiritual life — Bible reading (KJV, with highlights,
notes, and AI verse explanations), a digital hymnbook, and AI-generated
devotionals and prayers with scheduled reminders.

See [replit.md](./replit.md) for architecture, setup, and run instructions.

- `mobile/` — Expo (React Native) app
- `server/` — Node/Express API (OpenAI proxy, intended for Railway in production)
